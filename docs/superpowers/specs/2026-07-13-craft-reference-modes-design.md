# 写作模式参考类型调整

## 目标

- 新建写作模式只区分“小说”和“B站视频”。
- B站视频继续细分为“B站短篇故事”和“B站影视解说”。
- 移除新建页面、模式列表、故事创建选择器中的“鬼故事模式”展示。
- 影视解说参考用于后续制作原创短篇故事，而不是复制影视内容。

## 设计

1. `CraftMode` 增加 `bilibili-short-story` 和 `bilibili-commentary` 两个正式模式；`general` 继续作为小说模式，旧 `ghost-story` 仅保留读取兼容。
2. `CraftManager` 移除“通用/鬼故事”下拉框。小说直接使用 `general`；选择 B 站视频时显示两个子类型选项，默认 B 站短篇故事。
3. 写作模式卡片和详情标签按小说、B站短篇故事、B站影视解说显示，不再显示鬼故事标记；旧数据按来源显示，不影响打开和使用。
4. 核心分析提示词为两种 B 站子类型分别强调短篇故事节奏或影视解说的剧情压缩与原创改编边界。
5. 短篇故事创建使用影视解说模式时，自动补充“基于结构参考、重新设计原创短篇”的方向提示。

## 兼容与验证

- 服务端继续接受旧 `ghost-story` 请求和已保存 profile，但新请求只产生新模式值。
- 新增模式标签、payload、服务端归一化、核心提示词和短篇生成方向测试。
- 运行 Studio 相关测试、核心相关测试、类型检查、客户端构建，并在浏览器检查新建页面和写作模式选择器。
